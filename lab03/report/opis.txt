Opis HR
U ovom radu implementirala sam grafički prikaz algoritma višekriterijske optimizacije NSGA-II u Javi koristeći OpenGL (JOGL) biblioteku. Točnije, radi se o vizualizaciji samo dvokriterijske optimizacije zbog ograničenja s prikazom više dimenzija. Prikazano je kako se populacija rješenja mijenja kroz iteracije, postepeno postaje sve bolja i točnije opisuje optimalnu "funkciju". Različitim bojama prikazane su fronte populacije, gdje je crvenom bojom označena trenutno najbolja fronta. Zbog različitih kodomena kriterijskih funkcija, koordinatni sustav rješenja moguće je određenim tipkama skalirati i translatirati po obje osi odvojeno. Također, moguće je zaustaviti i pokrenuti izvođenje algoritma kako bi se vidjela populacija u određenom trenutku.

Opis ENG
In this project, I implemented a graphical representation of the NSGA-II multicriteria optimization algorithm in Java using the OpenGL (JOGL) library. More precisely, it is a visualization of only two-criteria optimization due to the limitation with the display of multiple dimensions. It is shown how the population of solutions changes through iterations, gradually becoming better and more accurately describing the optimal "function". Population fronts are shown in different colors, where the current best front is marked in red. Due to the different codomains of the criterion functions, the coordinate system of the solution can be scaled and translated along both axes separately using certain keys. Also, it is possible to stop and start the execution of the algorithm in order to see the population at a certain time.
